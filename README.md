# 🛒 Feira

App simples de lista de compras com controle de preços e quantidades.

## Funcionalidades

- Cola a lista do WhatsApp e o app separa os itens automaticamente
- Campo de quantidade e preço unitário por item
- Subtotal por item e total geral em tempo real
- Itens extras (coisas que você pega no mercado fora da lista)
- Barra de progresso dos itens marcados

## Tecnologia

HTML + CSS + JavaScript puro. Sem dependências, sem backend.

## Deploy

Publicado via GitHub Pages com deploy automático a cada push na branch `main`.
